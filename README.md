# RenameDirNameToFiles

*ディレクトリ名をファイル名に置換するバッチファイルです*

Windows10で動作確認済み

**動作イメージ**

置換前

| 第一階層 | 第二階層  | 
|-----------|-------------|
| TestDir | hoge.txt  |
|  | huga.bmp |
| Rename.bat |  |

置換後

| 第一階層 | 第二階層  |
|-----------|-------------|
| TestDir | TestDir.txt  |
|  | TestDir.bmp |
| Rename.bat |  |
